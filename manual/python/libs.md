# SeaTable脚本支持库

## Python3 标准库

支持 Python 标准库中的模块

## 第三方库

如下模块可供使用

### seatable_api

即本文档描述的SeaTable Python API, [seatable-api](https://pypi.org/project/seatable-api/)

### dateutils

日期/时间运算的工具库, [dateutils](https://pypi.org/project/dateutils/)

### requests

http请求库, [requests](https://pypi.org/project/requests/)
